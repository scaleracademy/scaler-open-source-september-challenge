## Gist 2: C++ Code Snippet

**Title**: Example of C++ Standard Template Library (STL)

**Description**: This Gist contains a code snippet in C++ showcasing the usage of a standard template library (STL) container.

**Code Snippet**:

```cpp
#include <iostream>
#include <vector>

int main() {
    // Example code using an STL container (e.g., std::vector)
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    
    for (const auto& num : numbers) {
        std::cout << num << " ";
    }
    
    return 0;
}
